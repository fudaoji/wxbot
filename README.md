- Memcached & Redis

#### 安装教程

1.  拉取项目
2.  在项目根目录下`cp env .env`, 修改.env对应的配置信息
3.  将项目目录下的install/install.sql导入数据库
4.  修改项目目录、runtime、public/uploads的读写权限
5.  根目录下执行 `composer update`
6.  默认超管账号：admin 密码：123456
